# PDF Merger 
A professional desktop application for merging PDF files with advanced features built with Python, CustomTkinter, and PyPDF.
## Features
- PDF Merging: Combine multiple PDF files into a single document
- Drag & Drop Reordering: Visually rearrange files with drag handles
- Up/Down Buttons: Precise file ordering with ▲▼ buttons
- File Previews: Thumbnails and detailed multi-page previews
- Page Range Selection: Merge specific pages from each PDF (e.g., "1,3,5-8")
- Password Support: Handle encrypted PDFs with password prompts
- Multiple Sorting Options: Sort by filename, date, size, or custom order
- Advanced Mode Toggle: Switch between simple and feature-rich interfaces
- Theme Support: Light, dark, and system theme options
- File Information: Display file size, page count, and modification dates
- Progress Tracking: Visual feedback during merge operations
## Installation
```bash
pip install -r requirements.txt
```
## Prerequisites
- Python 3.9+
- pip
## Start Virtual Environment
### BASH
```bash
source venv/Scripts/activate
```
### CMD
```cmd
venv\Scripts\activate
```
## Run Python
### BASH
```bash
python main.py
```
